01

The method

Anyone can prompt a page. auteur directs one — and refuses to ship slop.

  1. Commit-sheet

    One committed art direction — a single brand hue, a type system, a motion budget, named anti-references — written down before any markup. No "let's see how it looks" drift.

  2. Generated assets

    Imagery, video frames, depth maps and 3D geometry produced by local CLIs, routed by cost and by what each tool is actually good at.

  3. Build

    One WebGL context, GSAP/Lenis scroll, DOM motion on transform & opacity only — from proven recipes: fluid, particles, 3D worlds, scroll-morph.

  4. Gate

    Nothing ships until it clears slopscan (a runnable anti-slop linter) and motionqa (a real Playwright frame-rate & a11y check).
